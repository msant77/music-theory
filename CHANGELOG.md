# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-21

### Added

- Initial package scaffold
- Project structure with lib/, bin/, test/, example/ directories
- CLI entry point with --help and --version flags
- CI script for format, analyze, test, and coverage checks
- analysis_options.yaml with strict linting rules
- dart_test.yaml with unit and integration tags
- README.md with usage examples and installation instructions
- CLAUDE.md with development guidance

### Planned

- Note and pitch class implementation
- Interval calculations
- Chord construction and analysis
- Instrument definitions and tunings
- Chord voicing generation
- Transposition utilities
